⚙️ Transiciones en Tailwind CSS

🌀 Tipo de Transición

Las clases transition-* controlan qué propiedades del elemento deben animarse cuando hay un cambio. Es necesario combinar estas clases con clases hover: o focus: para ver sus efectos.

Clase Tailwind Equivalente CSS Descripción Ejemplo
transition-none transition: none; Desactiva todas las transiciones.
Sin transición
transition-all transition-property: all; Transición de todas las propiedades disponibles.
Todas las propiedades
transition transition-property: color, background-color, border-color, text-decoration-color, fill, stroke; Transición solo de propiedades comunes.
Colores y básicos
transition-colors transition-property: color, background-color, border-color, text-decoration-color, fill, stroke; Transición solo de colores.
Solo color
transition-opacity transition-property: opacity; Transición solo de la opacidad.
Solo opacidad
transition-shadow transition-property: box-shadow; Transición solo de la sombra del elemento.
Solo sombra
transition-transform transition-property: transform; Transición solo de transformaciones (rotate, scale, etc.).
Solo transform
transition-[border-radius] transition-property: border-radius; Transición solo de border-radius
Solo border-radius

🎯 Duración (duration-*)

Estas clases definen cuánto tiempo tarda una transición en completarse. Afecta directamente la velocidad del efecto.

Clase Tailwind Equivalente CSS Descripción Ejemplo
duration-75 transition-duration: 75ms; Transición muy rápida
Hover
duration-150 transition-duration: 150ms; Transición rápida
Hover
duration-300 transition-duration: 300ms; Duración estándar
Hover
duration-700 transition-duration: 700ms; Transición lenta
Hover
duration-1000 transition-duration: 1000ms; Transición muy lenta
Hover
duration-[450ms] transition-duration: 450ms; Valor personalizado
Hover

⏱️ Retardo (delay-*)

Las clases delay-* especifican cuánto tiempo espera una transición antes de comenzar. Se usa para crear efectos más dinámicos y secuenciales.

Clase Tailwind Equivalente CSS Descripción Ejemplo
delay-75 transition-delay: 75ms; Pequeño retardo
Hover
delay-150 transition-delay: 150ms; Retardo medio
Hover
delay-300 transition-delay: 300ms; Retardo largo
Hover
delay-700 transition-delay: 700ms; Retardo muy largo
Hover
delay-[250ms] transition-delay: 250ms; Valor personalizado
Hover

🎢 Curvas de transición (ease-*)

Las clases ease-* definen la curva de aceleración de las transiciones, controlando cómo se acelera o desacelera un cambio de estado.

Clase Tailwind Equivalente CSS Descripción Ejemplo
ease-linear transition-timing-function: linear; Transición constante, sin aceleración ni desaceleración
Hover
ease-in transition-timing-function: ease-in; Empieza lentamente y acelera al final
Hover
ease-out transition-timing-function: ease-out; Empieza rápido y desacelera al final
Hover
ease-in-out transition-timing-function: ease-in-out; Transición suave al inicio y al final
Hover
ease-[cubic-bezier(0.68,-0.55,0.27,1.55)] transition-timing-function: cubic-bezier(0.68,-0.55,0.27,1.55); Curva personalizada tipo "rebote"
Hover
