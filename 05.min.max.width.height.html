Clases Min/Max Width y Height en Tailwind

Clase CSS Ejemplo
min-w-16 min-width: 4rem;
Min Width
max-w-20 max-width: 5rem;
Max Width
min-h-20 min-height: 5rem;
Min Height
max-h-20 max-height: 5rem;
Max Height

Tailwind CSS utiliza la misma estructura de nombres de clases para definir restricciones mínimas y máximas de ancho y alto. Puedes usar min-w-*, max-w-*, min-h-* y max-h-* para establecer los valores mínimos y máximos del tamaño de un elemento. Estas clases aceptan los mismos valores que w-* y h-*, e incluso puedes usar valores personalizados como min-w-[150px] o max-h-[80vh].
