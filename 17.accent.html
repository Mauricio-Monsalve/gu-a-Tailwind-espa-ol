Clases de Accent en Tailwind CSS

Las clases accent-* permiten personalizar el color del acento para elementos como input[type=checkbox], input[type=radio] y progress.

Clase Tailwind CSS equivalente Ejemplo
accent-blue-500 accent-color: #3b82f6;
accent-pink-500 accent-color: #ec4899;
accent-[#ff9900] accent-color: #ff9900;
accent-green-500 accent-color: #22c55e;
accent-red-500 accent-color: #ef4444;
